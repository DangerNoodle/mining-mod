# Notice: This mod is defunct, now largely obsolete due to its features being included as part of vanilla content in mainline Bright Nights.

Formerly a mod made by @DangerNoodle, a few users including @Chaosvolt were added as maintainers to it long ago as they abandoned the project after disagreements with DDA's developers led to their removal from that project (context mostly present here: https://github.com/CleverRaven/Cataclysm-DDA/pull/21503). Since then, they hadn't returned to this project since, leaving it to be maintained by others complete with the development of a separate version for the fork Cataclysm: Bright Nights.

Its content has now since been pretty much entirely, save for a few minor things, been incorporated into vanilla gameplay as part of the Bright Nights fork instead (https://github.com/cataclysmbn/Cataclysm-BN). Due to the sole remaining maintainer's lack of interest in maintaining old mods for DDA, their frequent breaking changes, and lack of free time, this mod is now defunct. However, as ownership of the repo was never transferred over, and out of respect for the mod owner despite their absence, it can't and accordingly won't be deleted entirely.
